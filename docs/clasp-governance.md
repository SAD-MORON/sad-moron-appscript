# CLASP Governance for SAD-MORON-APPSCRIPT

- CLASP (Command Line Apps Script Projects) is optional and must be governed by SAD-MORON-FRAMEWORK.
- No credentials or .clasp.json files are to be committed to the repository.
- .clasp.json must be excluded from Git (see .gitignore).
- Local-only deployment linkage is permitted for development.
- No production binding or deployment without explicit framework approval.
- Script IDs are for reference only and must not be used for unauthorized deployment.
- VSCode workflow boundaries must be respected.
- Maintain separation between local repository and deployed Apps Script projects.
