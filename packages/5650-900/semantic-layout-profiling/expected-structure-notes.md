# Expected Institutional Structure Notes

These notes describe the kinds of institutional spreadsheet patterns this scaffold is expected to recognize. They are conceptual only and do not reproduce workbook content.

## Common Top-of-Sheet Pattern

- One or more merged title blocks near the first rows
- A compact institutional header zone beneath the title
- One or more administrative descriptor rows before the main table

## Common Middle Pattern

- A cuadro or table-label row introducing the next section
- Multi-row headers spanning several columns
- One or more visually dense table starts after separator rows

## Common Supporting Zones

- Observation or notes rows near the bottom of a table section
- Repeated administrative blocks across tabs or repeated sub-sections
- Fragmented visual regions separated by blank spacer rows

## Expected Output Style

- Zone types only
- Approximate row and column bounds
- Merge coordinates where relevant
- Confidence levels rather than extracted semantics

## Non-Goals

- Reconstructing docente records
- Reading institutional rosters
- Extracting DNI, CUIL, names, or personal identifiers
- Copying table contents into logs, evidence, or commits
